ğŸ“˜ Key Concept: What is a Variable?

ğŸ“– Understanding Variables

A variable is a named container that stores a value in your computer's memory. Think of it like a labeled box where you can put information.

Example:

student_name = "Alice"
student_age = 16
student_grade = 10

In this example, we created three "boxes" (variables) and put different pieces of information in them.

Class: class="info-key-concept"

ğŸ’¡ Variable Naming Best Practices

ğŸ’¡ Tips for Great Variable Names

  • Be descriptive: student_count not sc
  • Use snake_case: first_name not firstName
  • Avoid abbreviations: temperature not temp
  • Start with lowercase: name not Name
  • No reserved words: Don't use class, if, for, etc.

âœ… Good Example: Proper Variable Usage

âœ… Well-Named, Well-Structured Variables

# Student information
student_first_name = "Alice"
student_last_name = "Johnson"
student_age = 16
student_grade = 10
is_honor_roll = True

# Calculate and display

print(f"Honor Roll: {is_honor_roll}")

Why this is good:

  • Descriptive variable names
  • Follows snake_case convention
  • Uses appropriate data types
  • Well-organized and readable

Class: class="example-good"

âŒ Bad Example: Poor Variable Usage

âŒ Poorly-Named, Unclear Variables

# Unclear variable names

y = "Johnson"
z = 16
a = 10
b = True

# Confusing output

print(f"Age: {z}, Grade: {a}")

Why this is bad:

  • Single-letter names provide no context
  • Difficult to understand what each variable represents
  • Hard to maintain and debug
  • Not self-documenting

ğŸ“˜ Key Concept: Type Conversion

Python allows you to convert between types:

# String to integer
age = int("16")

# Integer to string
age_text = str(16)

# String to float
price = float("19.99")

ğŸ’¡ Type Conversion Tips

  • Always validate input before converting
  • Use try/except to handle conversion errors
  • Remember: "123" (string) â‰  123 (integer)

âš ï¸ Common Conversion Errors

Watch out for these mistakes:

  • int("hello") will cause an error
  • int(3.9) truncates to 3, doesn't round
  • Division always returns a float: 6 / 3 = 2.0
